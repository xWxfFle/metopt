## 1. Проверка раздела "Набросок: пересечение окружностей"

### 1.1. Вычитание ограничений

**Проверка:** Вычисление g₁ - g₂

```
g₁ - g₂ = (x² + y² - 1) - ((x-1)² + (y-1)² - 2)
        = x² + y² - 1 - (x² - 2x + 1 + y² - 2y + 1 - 2)
        = x² + y² - 1 - x² + 2x - 1 - y² + 2y - 1 + 2
        = 2x + 2y - 1
```

**Результат:** Правильно. Получено уравнение 2x + 2y = 1, откуда x + y = 1/2.

### 1.2. Подстановка в первое ограничение

**Проверка:** Использование x + y = 1/2 для получения xy = -3/8

Если x + y = 1/2, то y = 1/2 - x.

Подставляем в x² + y² = 1:
```
x² + (1/2 - x)² = 1
x² + 1/4 - x + x² = 1
2x² - x - 3/4 = 0

```

```

```

**Результат:** Правильно

### 1.3. Найденные точки пересечения

**Точки:**
